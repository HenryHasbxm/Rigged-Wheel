# Rigged Wheel

Background color is customizable. Wheel is not.
Names are put one on each line (return after each name to start a new section in the wheel)

Put a comma at the end of the name you want to win.
There is a lever in the bottom right to toggle on and off the rig. Off will revert the wheel to random and On will turn on the rig.
Please remember to turn on rigging and put the comma for the rig to work.

example:

Carter

Joseph

Henry,

Logan

                                Toggle On
